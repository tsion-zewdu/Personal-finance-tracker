# Personal Finance Tracker

A comprehensive Django-based personal finance tracking application with budget management, expense tracking, and financial reporting.

## Features

- **User Authentication**: Secure login, registration, and password reset
- **Transaction Management**: Add, edit, delete income and expenses
- **Category Management**: Custom categories with icons and budgets
- **Budget Tracking**: Set monthly budgets and get alerts
- **Financial Dashboard**: Visual overview with charts and KPIs
- **Reports**: Generate and download financial reports
- **CSV Export**: Export transaction data
- **User Profile**: Customize currency, timezone, and preferences
